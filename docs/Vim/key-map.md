# 快捷键

`i` 插前

`I` 插入最前

`a` 附后

`A` 附在最后

`o` 插入下一行

`O` 插入上一行

`G` 到最后一行

`gg` 到第一行

`yy` 复制当前行

`yw` 复制单词

`p` 粘贴

`dd` 删除当前行（实际更像剪切）

`dw` 删除单词

`cw` 改变单词

`ci{ or ci}` 改变大括号里的内容

`w` 下个单词首部

`e` 下个单词尾部

`b` 上个单词首部

`.` 重复上一次操作

`u` 撤销当前操作

`/` 搜索

`:%s/旧/新/g` 全局替换

`ctrl + r` 恢复前次操作

`esc` 正常模式

`:` 命令行模式

`:wq` 保存退出

`:q` 仅退出

`:q!` 不保存退出
