# 开发注意事项

## 敏感信息

客户的敏感信息需进行密文处理，业务员点击查看真实信息时需记录日志，并进行前台提示：

> 你的行为已经被系统记录，如果是有意收集客户资料是违法《网络安全法》的违法行为。情形严重者，可以入刑。如果正常工作行为，请忽略此条提醒

| 敏感信息 | 密文暴露方式      | 示例               |
| -------- | ----------------- | ------------------ |
| 手机号   | 隐藏中间四位 | 135\*\*\*\*8142        |
| 身份证号 | 隐藏后八位 | 4417021995\*\*\*\*\*\*\*\* |
