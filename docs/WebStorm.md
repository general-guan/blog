# WebStorm

## 快捷键

| 指令                 | 作用                     |
| -------------------- | ------------------------ |
| ctrl + alt + t       | 围绕包裹代码             |
| [ctrl + ] /          | 匹配 { } [ ]             |
| ctrl + shift + ] / [ | 匹配 { } [ ] 并选中      |
| ctrl + F12           | 显示当前文件结构         |
| ctrl + shift + F12   | 最大区域显示代码         |
| ctrl + r             | 替换                     |
| ctrl + shift + ↑ / ↓ | 行移动                   |
| ctrl + d             | 行复制                   |
| ctrl + - / +         | 折叠和展开代码块         |
| ctrl + ← / →         | 以单词作为边界跳光标位置 |
| ctrl + alt + l       | 格式化代码               |
| shift + enter        | 重新开始一行             |
| tab                  | 扩大缩进                 |
| shift + tab          | 减少缩进                 |
| ctrl + G             | 到指定行的代码           |
| alt + ↑ / ↓          | 上一个/下一个方法        |
